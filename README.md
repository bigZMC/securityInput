# securityInput
VUE安全支付组件

[在线DEMO](http://htmlpreview.github.io/?https://github.com/bigZMC/securityInput/blob/master/securityInput/dist/index.html)

# 运行步骤

```
1.cnpm install
2.cnpm run dev
```

# 组件说明
该组件使用vue完成web安全支付,目前只支持数字,a-z,A-Z的输入
###### 传入参数
length: 密码框位数,默认为6
###### 传出函数
securityInput: 对象数组,value代表每一位的输入值
